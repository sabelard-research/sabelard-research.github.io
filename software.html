Software

  • Ongoing participation to a Mathemagix library to compute Riemann-Roch spaces of smooth divisors on ordinary curves.
  • The code designed with P. Gaudry and P.-J. Spaenlehauer to experiment our point-counting algorithm for genus-3 hyperelliptic curves with explicit real multiplication can be found here.

Record computations

  • Using the above code, P.~Gaudry, P.-J. Spaenlehauer and I counted points on a genus-3 hyperelliptic curve with explicit RM defined over a 64-bit prime field. This gives a 192-bit Jacobian, which is a record size.
